### Setup (if not done already):
  1. Save the script as gcommit.py.
  2. Make it executable:
        ```chmod +x gcommit.py```
  3. Move it to /usr/local/bin for global access:
        ```sudo mv gcommit.py /usr/local/bin/gcommit```
